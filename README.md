# Mamachef
## Your friend in the kitchen!

Mamachef is a web application designed to help users manage their recipes and shopping lists. It features a clean and responsive design, making it easy to use on both desktop and mobile devices.

I chose this project for my final assignment in CS50 Web Development because I wanted to create something not common, that could be useful in everyday life, has a grade of complexity, and could then be expanded in the future with AI features.

This project, in particular idea, requirements and backend structure, is inspired and based on the project with the same name created by myself and my mates for university course of Software Design and Development, originally realized in Java with Spring Boot. I thank my friends Samuele and Stefano for letting me use the project as a base for this new version for CS50

Stack:
- Django: Backend framework
- Tailwind CSS: Styling framework
- Django Cotton: Component-based templating
